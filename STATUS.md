## 六、技术备注（给以后自己看）

- 网站框架：Ebisu EC（Java Web），session 靠 `JSESSIONID` cookie 维持
- 登录前必须先 GET 登录页拿 cookie，否则报"需要开启 cookie"
- 加购物车的关键：`request=insert`（JS 里 `putItemToCart()` 设的）
- Python requests 要加 `verify=False` 绕过本地代理的 SSL 拦截
- 503 是网站限流（不是被封），加 3-6 秒延迟 + 重试就能过
- 一个 session 内所有请求共享同一个 `token`
